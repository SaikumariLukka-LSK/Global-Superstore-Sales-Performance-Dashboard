# Global Superstore Sales & Performance Dashboard

## Project Overview
This project presents a comprehensive Business Intelligence solution developed using Microsoft Power BI based on the Global Superstore 2016 dataset. The dashboard provides actionable insights into sales performance, profitability, customer segments, product categories, shipping efficiency, and return analysis.

The data was cleaned, transformed, and modelled using a Star Schema to support efficient reporting and interactive visualizations.

## Objectives
- Identify the most profitable regions and product categories.
- Analyze customer segments contributing the highest revenue and profit.
- Evaluate shipping modes and their impact on profitability.
- Monitor product return rates and identify frequently returned products.
- Support data-driven business decision-making through interactive dashboards.

## Tools & Technologies
- Microsoft Power BI
- Power Query (M Language)
- DAX (Data Analysis Expressions)
- Star Schema Data Modelling
- Microsoft Excel

## Dataset
**Source:** Global Superstore 2016 Dataset

The dataset contains:
- 10,000+ records
- Sales, Profit, Quantity, Discount, and Shipping data
- Customer, Product, Region, and Order information

## Data Preparation
- Removed unnecessary columns
- Handled missing values
- Converted data types
- Created dimension tables
- Built a Star Schema data model
- Developed DAX measures and calculated columns

## Key Insights
- Technology is the most profitable product category.
- Corporate customers generate significant revenue.
- The West region records high sales but lower profit margins.
- Standard Class is the most used shipping mode.
- Return rate is approximately 2.1%.
- Sales show strong seasonal growth patterns.

## Business Recommendations
- Expand operations in high-performing regions.
- Optimize shipping costs to improve profitability.
- Reduce returns through product quality improvements.
- Review discount strategies to maintain healthy margins.
- Focus on high-value customer segments.
